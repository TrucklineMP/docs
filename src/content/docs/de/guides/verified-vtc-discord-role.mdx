# Verifizierte VTC-Discord-Rolle

Verifizierte VTC-Inhaber können eine spezielle Rolle auf dem **Haupt-Discord-Server von TrucklineMP** beantragen: **Verifizierter VTC-Manager/Inhaber**. Diese Rolle ist für die Leitung einer VTC auf dem offiziellen Community-Server gedacht. Sie ist unabhängig vom eigenen Discord-Server deiner VTC und von der [VTC-Discord-Server-Verifizierung](/de/guides/vtc-discord-verification/) in deinem Profil.

Das System ist vollständig automatisiert. Anfragen werden anhand deines TrucklineMP-Kontos und deiner VTC-Berechtigungen geprüft. Für eine gültige Anfrage ist keine manuelle Genehmigung durch das Team erforderlich.

---

## Was du erhältst

- Die Rolle **Verifizierter VTC-Manager/Inhaber** auf dem Haupt-Discord von TrucklineMP
- Anerkennung als Leitung einer [Verifizierten VTC](/de/guides/verified-vtc-program/)
- Bis zu **zwei** Rolleninhaber pro VTC:
  - Der **VTC-Inhaber** (erforderlicher Platz)
  - **Ein optionaler Manager** deiner VTC

---

## Voraussetzungen

Bevor du die Rolle beantragst, stelle sicher, dass alle folgenden Punkte erfüllt sind.

### Für den VTC-Inhaber (dich)

| Voraussetzung                     | Details                                                                                                                                 |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Verifizierte VTC**              | Deine VTC muss den genehmigten Status des [Verifizierungsprogramms für VTCs](/de/guides/verified-vtc-program/) auf TrucklineMP besitzen |
| **Inhaberschaft**                 | Du musst der VTC-Inhaber sein. Nur der Inhaber kann die Anfrage stellen                                                                 |
| **Verknüpfter Discord**           | Dein TrucklineMP-Konto muss mit dem Discord-Konto verbunden sein, das du auf dem Haupt-Server verwendest                                |
| **Verifizierte verknüpfte Rolle** | Du musst die **Verified** [Linked Role](/de/guides/discord-linked-roles/) auf dem Haupt-Discord-Server besitzen                         |
| **Mitglied des Haupt-Servers**    | Du musst Mitglied des offiziellen TrucklineMP-Discords sein                                                                             |

### Für den optionalen Manager

Wenn du eine zweite Person ernennst, muss diese ebenfalls:

- Ein **Mitglied** deiner VTC auf TrucklineMP sein
- Eine **Management**-Position in dieser VTC besitzen (die Rolle mit VTC-Verwaltungsberechtigungen)
- Ihr TrucklineMP-Konto **mit Discord verknüpft** haben
- Die **Verified** Linked Role auf dem Haupt-Discord-Server besitzen
- Diese Rolle nicht bereits für eine andere VTC besitzen

Du kannst dich nicht selbst zweimal ernennen. Der Inhaber-Platz und der Manager-Platz sind getrennt.

---

## So beantragst du die Rolle

Anfragen werden auf dem Haupt-Discord von TrucklineMP im dafür vorgesehenen **Verified VTC role**-Kanal gestellt (veröffentlicht von Modbot).

1. Tritt dem [offiziellen TrucklineMP Discord-Server](/de/guides/official-discord-server/) bei.
2. Führe den [Discord Linked Roles](/de/guides/discord-linked-roles/)-Prozess durch und beanspruche die **Verified**-Rolle (**Servername → Linked Roles**).
3. Öffne den Anfragekanal für die Verified VTC-Rolle.
4. Klicke in der Panel-Nachricht auf **Request Role**.
5. Im Formular:
   - Lasse das Manager-Feld **leer**, um die Rolle nur dir selbst zuzuweisen, oder
   - Gib die **Discord-Benutzer-ID** eines VTC-Managers an, den du ernennen möchtest (numerische ID, kein Anzeigename).
6. Sende das Formular ab. Du erhältst eine private, nur für dich sichtbare Antwort mit dem Ergebnis.

Wenn beide Plätze vergeben werden, erhalten du und dein ernannter Manager die Rolle innerhalb weniger Sekunden.

### Später einen Manager ernennen oder ändern

Sende eine neue Anfrage mit einer anderen Discord-Benutzer-ID des Managers. Das System ersetzt den aktuellen Manager-Platz, falls dieser bereits vergeben ist. Dein Inhaber-Platz wird bei jeder Anfrage aktualisiert.

### Anfrage-Limit

Du kannst pro Discord-Konto eine Anfrage pro Minute senden.

---

## Status auf TrucklineMP anzeigen

VTC-Inhaber können sehen, wer die Rolle besitzt:

1. Öffne deine VTC-**Verwaltungskonsole**.
2. Gehe zu **Settings**.
3. Scrolle zu **Main Discord Role**.

Dieser Bereich zeigt:

- Ob deine VTC verifiziert ist
- Wer die **Inhaber- und Manager-Plätze** besitzt (falls vergeben)
- Einen Link zum Anfragekanal auf Discord (wenn eingerichtet)

Nur der VTC-Inhaber kann diesen Bereich sehen.

---

## Wenn die Rolle automatisch entfernt wird

Die Rolle wird überwacht. Sie wird sofort entfernt, sobald die Voraussetzungen nicht mehr erfüllt sind. Du behältst die Discord-Rolle nicht, wenn deine VTC oder deine Position nicht mehr berechtigt ist.

Häufige Gründe für die Entfernung:

| Ereignis                                           | Was passiert                                                                                              |
| -------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| VTC verliert den **verifizierten** Status          | Alle Vergaben für diese VTC werden widerrufen                                                             |
| VTC wird **aufgelöst**                             | Alle Vergaben für diese VTC werden widerrufen                                                             |
| **Übertragung der Inhaberschaft**                  | Die Vergabe für den vorherigen Inhaber wird widerrufen. Der neue Inhaber muss die Rolle erneut beantragen |
| Mitglied wird aus der VTC **entfernt**             | Die Vergabe für dieses Mitglied wird widerrufen                                                           |
| Manager verliert **Verwaltungsberechtigungen**     | Die Vergabe für diesen Manager wird widerrufen                                                            |
| **Discord wird von TrucklineMP getrennt**          | Alle Vergaben für diesen Benutzer werden widerrufen                                                       |
| **Verified** Linked Role wird auf Discord entfernt | Die Vergabe wird bei der nächsten Berechtigungsprüfung widerrufen                                         |

Nach dem Verlust des Verifizierungsstatus gibt es keine Übergangsfrist. Beantrage die Rolle erst erneut, wenn deine VTC wieder verifiziert ist und du weiterhin alle Voraussetzungen erfüllst.

## Fehlerbehebung

### "You must own a verified VTC"

Deine VTC besitzt keinen aktiven Verified-Programm-Status oder du bist nicht der Inhaber. Überprüfe den **Verified Status** in deinem Verwaltungspanel.

### "You need the Verified linked role"

Beanspruche die **Verified**-Rolle über **Servername → Linked Roles** auf dem Haupt-Discord. Die Verbindung von Discord unter trucklinemp.com **Settings → Connections** allein erfüllt diese Voraussetzung nicht.

### "The nominated user must have a VTC Management position"

Die eingegebene Person ist kein VTC-Mitglied mit Verwaltungsberechtigungen. Weise ihr zuerst die Rolle **Management** (oder entsprechende Berechtigungen) in deiner VTC zu.

### "The nominated manager must have the Verified linked role"

Dein Manager muss zuerst die Linked-Role-Option auf dem Haupt-Server abschließen, bevor du ihn ernennen kannst.

### "This user already holds the role for another VTC"

Jede Person kann diese Rolle nur für eine VTC gleichzeitig besitzen.

### "You own multiple verified VTCs"

Der automatisierte Prozess unterstützt eine verifizierte VTC pro Inhaber. Wende dich an das TrucklineMP-Team, wenn du Hilfe bei einem Sonderfall benötigst.

### Rolle fehlt nach erfolgreicher Anfrage

- Stelle sicher, dass du dich auf dem **Haupt**-TrucklineMP-Discord befindest und nicht auf einem Staff- oder Beta-Server.
- Warte einige Sekunden und überprüfe deine Rollen erneut.
- Wenn das Problem weiterhin besteht, öffne ein [Support-Ticket](/guides/support-tickets/) mit deinem VTC-Namen und Discord-Benutzernamen.

## Unterschiede zu anderen Discord-Funktionen

| Funktion                                                                 | Zweck                                                                                              |
| ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------- |
| [Discord Linked Roles](/de/guides/discord-linked-roles/)                 | Optionale **Verified**-Rolle für jedes Mitglied mit verknüpftem TrucklineMP-Konto                  |
| [VTC-Discord-Server-Verifizierung](/de/guides/vtc-discord-verification/) | Verknüpft **den Discord-Server deiner VTC** mit deinem TrucklineMP-VTC-Profil                      |
| **Verifizierter VTC-Manager/Inhaber** (diese Anleitung)                  | Leitungsrolle auf dem **Haupt**-TrucklineMP-Discord für verifizierte VTC-Inhaber und einen Manager |

Du benötigst die **Verified** Linked Role, bevor du die Rolle Verifizierter VTC-Manager/Inhaber beantragen kannst.
