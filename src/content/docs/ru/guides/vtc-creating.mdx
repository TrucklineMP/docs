# Создание VTC

Виртуальная транспортная компания (VTC) это дом вашей организации в TrucklineMP. Это руководство рассказывает о требованиях к кандидатам, мастере создания и о том, что происходит после запуска вашей VTC.

## Перед началом

Вам потребуется:

- Авторизованный аккаунт TrucklineMP
- Разрешение на создание VTC (применяются правила соответствия требованиям платформы)
- Отсутствие существующей VTC, где вы уже являетесь Основателем (одна VTC на аккаунт)

Откройте процесс создания по адресу [trucklinemp.com/vtc/create](https://trucklinemp.com/vtc/create).

---

## Проверка соответствия требованиям

Когда вы открываете страницу создания, TrucklineMP автоматически запускает проверку соответствия требованиям. Если вы не соответствуете требованиям, страница объяснит причину. Общие причины включают:

- Уже являетесь Основателем VTC
- Репутация аккаунта или ограничения платформы
- Функция отключена настройками платформы

Вы не сможете продолжить, пока не пройдете проверку соответствия требованиям.

---

## Мастер создания

Мастер состоит из пяти шагов: **Идентификация**, **Описание**, **Правила**, **Детали** и **Проверка**.

### Шаг 1: Идентификация

| Поле | Примечания |
|-------|-------|
| **VTC Name** | Отображаемое имя, видимое на всей платформе. Обязательно. |
| **Handle** | Уникальный URL-адрес: `trucklinemp.com/vtc/@yourhandle`. От 2 до 15 символов, только строчные буквы, цифры и подчеркивания. Обязательно. |
| **Slogan** | Краткий слоган (необязательно). |
| **Primary Language** | Язык по умолчанию для вашей VTC. |
| **Logo** | URL-адрес HTTPS на квадратное изображение профиля (необязательно). |
| **Banner** | URL-адрес HTTPS на широкое изображение обложки (необязательно). |
| **Visibility** | Режимы public, unlisted или private. См. [Видимость](/ru/guides/vtc-visibility/). |

### Шаг 2: Описание

Напишите историю вашей VTC с использованием Markdown. Минимум 20 символов. Это отображается в вашем публичном профиле.

### Шаг 3: Правила

Напишите правила для участников с использованием Markdown. Минимум 20 символов. Кандидаты и участники смогут прочитать их в вашем профиле.

### Шаг 4: Детали

| Поле | Примечания |
|-------|-------|
| **Minimum Account Age** | Количество дней, которое должен существовать аккаунт TrucklineMP перед подачей заявки. Установите `0` для отсутствия ограничений. |
| **Minimum Playtime** | Количество часов игрового времени, необходимое для подачи заявки. Установите `0` для отсутствия ограничений. |
| **Recruitment Open** | Могут ли кандидаты подавать заявки сразу после создания. |
| **Contact Email** | Необязательный публичный контактный адрес. |
| **Social Links** | Веб-сайт, приглашение Discord, Instagram, Facebook, X/Twitter. Если указаны, должны быть полными URL-адресами `https://`. |

### Шаг 5: Проверка

Подтвердите каждое поле, затем отправьте. В случае успеха вы будете перенаправлены в профиль вашей новой VTC.

## Настройка по умолчанию

Когда ваша VTC создана, TrucklineMP устанавливает базовые роли:

- **Management**
- **Staff**
- **Driver**
- **Recruit**

Вы становитесь Основателем с полными правами. Откройте **Manage** в профиле вашей VTC, чтобы настроить набор персонала, участников и привязку Discord.

Рекомендуемые следующие шаги:

1. [Общие настройки](/ru/guides/vtc-general-settings/) - уточните брендинг и социальные ссылки
2. [Роли и права](/ru/guides/vtc-roles-permissions/) - настройте вашу иерархию

## Устранение неполадок

### "Вы не соответствуете требованиям для создания VTC"

Прочтите причины, указанные на странице создания. Возможно, вы уже являетесь Основателем VTC или вам нужно решить проблему с репутацией.

### Имя пользователя не сохраняется

Имена пользователей должны содержать от 2 до 15 символов, быть в нижнем регистре и использовать только буквы, цифры и подчеркивания. Имя пользователя должно быть уникальным.

### Ошибка создания из-за социальных URL-адресов

Каждый необязательный URL-адрес должен начинаться с `https://`. Ссылки-приглашения Discord должны быть полными URL-адресами, а не просто кодами.
