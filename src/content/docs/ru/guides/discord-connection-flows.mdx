# Варианты подключения Discord

TrucklineMP использует Discord несколькими способами. Они решают разные задачи и **не** заменяют друг друга.

| Вариант | Где | Цель |
|------|-------|---------|
| **Подключение аккаунта** | Настройки сайта | Идентификация, восстановление, личные сообщения с уведомлениями |
| **Связанные роли** | Сервер Discord | Роли по желанию, регулируемые правилами TrucklineMP |
| **Верификация сервера VTC** | Discord `/verify-server` | Подтверждение того, что сервер принадлежит вашей VTC на сайте |
| **Роль Verified VTC** | Главный Discord TrucklineMP | Руководящая роль для основателей верифицированных VTC |

---

## 1. Подключение аккаунта

**Путь:** trucklinemp.com → Settings → **Connections** → Connect Discord

**Использовать для:**

- Привязки Discord к вашей личности TrucklineMP
- Вариантов восстановления аккаунта
- [Личных сообщений с уведомлениями Discord](/ru/guides/discord-notifications/)

**Не делает:**

- Не выдает связанные роли на сервере Discord
- Не верифицирует сервер VTC в вашем профиле
- Не добавляет роли автоматически где-либо

Вы можете отключиться и подключиться заново в разделе Connections, если нет ограничений от персонала.

---

## 2. Связанные роли (Сервер Discord)

**Путь:** Discord → Меню сервера → **Linked Roles** → подключить TrucklineMP

**Использовать для:**

- Администраторов серверов, требующих от участников доказать наличие аккаунта TrucklineMP
- Дополнительной проверки на принадлежность к **персоналу TrucklineMP** для приватных каналов
- Закрытия доступа к каналам за верифицированной ролью

**Не делает:**

- Не заменяет привязку в Connections на сайте для личных сообщений (хотя должен использоваться тот же аккаунт Discord)
- Не отображает значок VTC на trucklinemp.com
- Не запускает `/verify-server` для подтверждения владения VTC

Участники должны **самостоятельно соглашаться** на каждом сервере. Роли не назначаются скрыто.

Полная настройка: [Связанные роли](/ru/guides/discord-linked-roles/)

---

## 3. Верификация сервера VTC

**Путь:** VTC Manage → General Settings → сгенерировать токен → Discord `/verify-server`

**Использовать для:**

- Отображения официального Discord вашей VTC в вашем профиле TrucklineMP
- Использования `/vtc` на этом сервере для отображения информации о компании
- Подтверждения владения сервером для посетителей сайта

**Требует:**

- Основателя VTC или персонал с правом настройки Discord
- Установленного бота TrucklineMP на сервере
- Прав администратора в Discord для запуска `/verify-server`

**Не делает:**

- Не настраивает Связанные роли (делайте это отдельно в настройках ролей Discord)
- Не привязывает личные аккаунты участников, если они также не используют Связанные роли или Подключения

Полная настройка: [Верификация Discord VTC](/ru/guides/vtc-discord-verification/)

---

## 4. Роль Verified VTC (главный сервер)

**Путь:** Главный Discord TrucklineMP → канал роли Verified VTC → **Request Role**

**Использовать для:**

- Основателей верифицированных VTC (и одного менеджера по желанию), которые хотят получить руководящую роль в **официальном** Discord TrucklineMP
- Признания отдельно от Discord-сервера вашей собственной VTC

**Требует:**

- Статуса в [Программе верифицированных VTC](/ru/guides/verified-vtc-program/)
- **Верифицированной** [связанной роли](/ru/guides/discord-linked-roles/) на главном сервере
- Основатель VTC подает запрос через Modbot (необязательный ID пользователя Discord менеджера)

**Не делает:**

- Не верифицирует Discord-сервер вашей VTC в профиле (используйте вариант 3)
- Не заменяет Связанные роли на серверах сообщества

Полная настройка: [Роль Verified VTC в Discord](/ru/guides/verified-vtc-discord-role/)

## Один аккаунт Discord для всего

Используйте один аккаунт Discord для Подключений, Связанных ролей и верификации. Несоответствующие аккаунты вызывают ошибки "не привязан" в `/profile` и сбои при проверке ролей.
