# Каталог VTC

Каталог VTC помогает водителям находить виртуальные транспортные компании. Просматривайте все публичные списки, фильтруйте по важным для вас параметрам и открывайте профиль VTC, чтобы подать заявку или узнать больше.

## Доступ к каталогу

Откройте [trucklinemp.com/vtc](https://trucklinemp.com/vtc).

Здесь отображаются только **публичные** и **скрытые** из списков VTC. Приватные VTC скрыты, если у вас нет ссылки-приглашения.

---

## Поиск и фильтры

Используйте панель инструментов для сужения результатов:

| Фильтр | Назначение |
|--------|---------|
| **Поиск** | Совпадение по названию или имени пользователя VTC |
| **Верифицированные** | Показать только верифицированные VTC |
| **Официальные** | Показать официальные VTC платформы |
| **Набор открыт** | VTC, которые в данный момент принимают заявки |
| **Язык** | Фильтр основного языка |

### Параметры сортировки

- Сначала новые
- По названию (А-Я)
- По количеству участников

Результаты обновляются при изменении фильтров. Пагинация загружает больше VTC, если они доступны.

## Карточки VTC

На каждой карточке отображается:

- Превью логотипа и баннера
- Наименование VTC и название (`@handle`)
- Слоган или краткая выдержка из описания
- Значки (верифицирована, официальная, партнерская, набор открыт)
- Количество участников
- Основной язык

Нажмите на карточку, чтобы открыть полный профиль VTC.

---

## Обзор профиля VTC

Из профиля вы можете:

- Прочитать полное описание и правила
- Посмотреть предстоящие мероприятия и новости
- Посмотреть превью участников и открыть полный список
- Подать заявку, если набор открыт
- Присоединиться по приглашению, если у вас есть код
- Увидеть ссылку на верифицированный сервер Discord, если он настроен

В URL-адресах профилей используются имена пользователей: `trucklinemp.com/vtc/@handle`

---

## Для Основателей VTC

Чтобы улучшить видимость:

1. Установите видимость на публичную в [Общих настройках](/ru/guides/vtc-general-settings/)
2. Держите набор открытым, если вы ищете сотрудников
3. Подайте заявку на статус [Верифицированной](/ru/guides/verified-vtc-program/) или [Партнерской](/ru/guides/partnered-vtc-program/), если соответствуете требованиям
4. Публикуйте [новости](/ru/guides/vtc-news/) и [мероприятия](/ru/guides/vtc-events/), чтобы ваш профиль оставался активным

Скрытые из списков VTC доступны по прямой ссылке, но не отображаются в результатах поиска каталога.

## Доступ к API

Сторонние инструменты могут выводить список VTC через публичный API:

```
GET https://api.trucklinemp.com/vtcs
```

См. [Публичный API](/ru/guides/developers/public-api/) для получения информации об аутентификации и пагинации.
