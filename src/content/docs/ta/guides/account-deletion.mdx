# Account Deletion

உங்கள் TrucklineMP கணக்கை நிரந்தரமாக நீக்குவது மற்றும் எந்தத் தரவுகள் அகற்றப்படுகின்றன என்பதை அறியவும்.

## Account Deletion-ஐ அணுகுவது

1. மேல் வலது மூலையில் உள்ள உங்கள் Avatar-ஐ கிளிக் செய்யவும்.
2. **Settings** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Danger** என்பதைத் தேர்ந்தெடுக்கவும்.
4. அல்லது `trucklinemp.com/user/me/settings?tab=danger` முகவரிக்குச் செல்லவும்.

---

## கணக்கை நீக்கும் முன்

### இந்த மாற்று வழிகளைப் பரிசீலிக்கவும்
- **சிறிது இடைவெளி எடுத்துக்கொள்ளுங்கள்** — உங்கள் கணக்கை நீக்காமல் TrucklineMP-ஐ பயன்படுத்தாமல் இருக்கலாம்.
- **உங்கள் தரவை ஏற்றுமதி செய்யுங்கள்** — கணக்கை நீக்கும் முன் Data Export-ஐ கோருங்கள்.
- **வெளிப்புற கணக்குகளின் இணைப்பை நீக்குங்கள்** — மீண்டும் பதிவு செய்ய திட்டமிட்டிருந்தால், இணைக்கப்பட்ட கணக்குகளை நீக்கலாம்.

### முக்கிய எச்சரிக்கைகள்
- **இந்தச் செயல் நிரந்தரமானது; இதை மீண்டும் மாற்ற முடியாது.**
- உங்கள் அனைத்து தரவுகளும் நிரந்தரமாக நீக்கப்படும்.
- கணக்கை நீக்கிய பிறகு அதை மீட்டெடுக்க முடியாது.
- உங்கள் Username மற்றும் Handle பிற பயனர்களுக்குக் கிடைக்கக்கூடும்.
- Two-Factor Authentication (2FA) செயல்படுத்தப்பட்டிருந்தால், கணக்கை நீக்குவதை உறுதிப்படுத்த Verification Code அல்லது Backup Code-ஐ உள்ளிட வேண்டும்.

---

## என்னென்ன நீக்கப்படும்?

### நிரந்தரமாக அகற்றப்படும்
- உங்கள் Profile (பெயர், Handle, Profile Picture, சமூக வலைதள இணைப்புகள்)
- அனைத்து VTC உறுப்பினர் பதிவுகளும் மற்றும் பதவிகளும்
- உங்கள் செயல்பாட்டு வரலாறு
- Forum பதிவுகள் மற்றும் Threads
- Event பங்கேற்பு பதிவுகள்
- Configuration தரவுகள்
- உங்களைப் பற்றிய Staff குறிப்புகள்
- Session தரவுகள்
- இணைக்கப்பட்ட கணக்குத் தகவல்கள்

### தொடர்ந்து இருக்கக்கூடியவை
- பகுப்பாய்வு (Analytics) நோக்கத்திற்காக சில பெயரற்ற (Anonymized) தரவுகள் வைத்திருக்கப்படலாம்.
- சட்டத் தேவைகளுக்கிணங்க சில பதிவுகள் பாதுகாக்கப்படலாம்.

## உங்கள் கணக்கை நீக்குவது எப்படி?

### படி 1: நீக்கும் செயல்முறையைத் தொடங்குதல்
1. **Settings** > **Danger** தாவலுக்குச் செல்லவும்.
2. **Delete Account** என்பதைக் கிளிக் செய்யவும்.
3. உறுதிப்படுத்தும் Modal தோன்றும்.

### படி 2: Countdown முடியும் வரை காத்திருக்கவும்
- 5 விநாடி Countdown தொடங்கும்.
- இது தவறுதலாக கணக்கை நீக்குவதைத் தடுக்கிறது.
- Countdown முடியும் வரை அடுத்த படிக்குச் செல்ல முடியாது.

### படி 3: நீக்குவதை உறுதிப்படுத்துதல்
1. உறுதிப்படுத்தும் பெட்டியில் `DELETE` என்று துல்லியமாக உள்ளிடவும்.
2. Countdown முடிந்த பிறகு Delete Button செயல்படும்.
3. 2FA செயல்படுத்தப்பட்டிருந்தால், உங்கள் Authenticator Code அல்லது Backup Code-ஐ உள்ளிடுமாறு கேட்கப்படும்.
4. **Delete My Account** என்பதைக் கிளிக் செய்யவும்.

### படி 4: செயல்முறை நிறைவு
- உங்கள் கணக்கு உடனடியாக நீக்கப்படும்.
- நீங்கள் அனைத்து அமர்வுகளிலிருந்தும் வெளியேற்றப்படுவீர்கள்.
- முகப்புப் பக்கத்திற்கு (Homepage) திருப்பி அனுப்பப்படுவீர்கள்.
- வெற்றிகரமாக நீக்கப்பட்டதை உறுதிப்படுத்தும் அறிவிப்பு காட்டப்படும்.

## கணக்கை நீக்கிய பிறகு

### என்ன நடைபெறும்?
- அனைத்து Sessions-லிருந்தும் நீங்கள் வெளியேற்றப்படுவீர்கள்.
- உங்கள் Profile Page 404 பிழையைக் காட்டும்.
- உங்கள் அனைத்து VTC உறுப்பினர் பதிவுகளும் அகற்றப்படும்.
- உங்கள் Forum பதிவுகள் மற்றும் Threads, மற்ற தரவுகளுடன் சேர்த்து நீக்கப்படும்.

### நான் மீண்டும் பதிவு செய்ய முடியுமா?
- ஆம். அதே Email முகவரியைப் பயன்படுத்தி புதிய கணக்கை உருவாக்கலாம்.
- உங்கள் பழைய தரவுகள் மீட்டெடுக்கப்படாது.
- உங்கள் Username மற்றும் Handle மீண்டும் கிடைக்கக்கூடும்.
- VTC உரிமையை (Ownership) மீட்டெடுக்க முடியாது.

- உங்கள் Privacy Settings-ஐ மாற்றவும்.

### புதிய தொடக்கம் வேண்டுமெனில்
- உங்கள் Display Name மற்றும் Handle-ஐ மாற்றவும்.

- முதலில் உங்கள் தரவை Export செய்து வைத்துக்கொள்ளவும்.

### தனியுரிமை குறித்து கவலை இருந்தால்
