# பொது சுயவிவரம்

TrucklineMP-இல் மற்ற ஓட்டுநர்கள் (Drivers) உங்களைப் பார்க்கும் பொதுவான அடையாளம் (Public Identity) தான் உங்கள் Public Profile. இதில் உங்கள் அடையாளம், VTC உறுப்பினர் விவரங்கள், சமூக செயல்பாடுகள் மற்றும் உங்கள் கணக்கின் நிலை (Standing) ஆகியவை ஒருங்கிணைக்கப்பட்டிருக்கும்.

## சுயவிவரம்-ஐ திறப்பது

- **உங்கள் சுயவிவரம்:** உங்கள் Avatar-ஐ கிளிக் செய்யவும் அல்லது `trucklinemp.com/user/me` முகவரிக்குச் செல்லவும் (இது உங்கள் Handle அல்லது WebID URL-க்கு தானாக மாற்றப்படும்).
- **மற்றொரு பயனரின் சுயவிவரம்:** `trucklinemp.com/@handle` அல்லது `/user/{webId}` முகவரியைப் பயன்படுத்தவும்.
- [User Directory](/guides/user-directory/) வழியாகவும் சுயவிவரம்-ஐ அணுகலாம்.

---

## Profile header

Profile Header-இல் பின்வரும் தகவல்கள் காண்பிக்கப்படும்:

- Avatar மற்றும் Display Name
- Handle (`@username`)
- WebID
- Staff அல்லது Program Badges
- செயலில் இருக்கும் Ban அல்லது Suspension Banner (தேவையானபோது)
- இணைக்கப்பட்டு Visible ஆக இருந்தால் நேரடி **Twitch** அல்லது **YouTube** நிலை (Live Status)
- Quick Actions (உள்நுழைந்த பயனர்களுக்கு **Report User** வசதி)

## Profile tabs

ஒவ்வொரு கணக்கிலும் காணப்படும் Tabs மாறுபடலாம். பொதுவாக காணப்படும் பகுதிகள்:

### Overview

உங்கள் Summary Statistics, Roles, Badges மற்றும் முக்கிய தகவல்களின் சுருக்கம்.

### VTC

தற்போதைய VTC உறுப்பினர் விவரம், உங்கள் Role மற்றும் அந்த VTC Profile-க்கான இணைப்பு. எந்த VTC-யிலும் உறுப்பினராக இல்லையெனில், இந்தப் பகுதியில் காலியான நிலை (Empty State) காண்பிக்கப்படும்.

### Activity

சமீபத்திய Platform செயல்பாடுகள் (Posts, Event Participation மற்றும் கிடைக்கக்கூடிய பிற செயல்பாடுகள்) காண்பிக்கப்படும்.

### Forum

Forum-இல் உங்கள் செயல்பாடுகளின் சுருக்கம் மற்றும் Forum Module செயல்படுத்தப்பட்டிருந்தால் [forum.trucklinemp.com](https://forum.trucklinemp.com)-இல் உள்ள தொடர்புடைய பதிவுகளுக்கான இணைப்புகள்.

### Configs

நீங்கள் பொதுவாகப் பகிரத் தேர்ந்தெடுத்துள்ள External Truck Configurations.

**Settings**-இல் **Hide External Configs** என்பதை இயக்கியிருந்தால், இந்தப் பகுதி மற்ற பயனர்களுக்குக் காண்பிக்கப்படாது.

### Record

Standing Rules-இன் அடிப்படையில் உங்கள் Ban மற்றும் Moderation வரலாறு. உங்கள் சொந்த Profile-இல், பிற பயனர்களைக் காட்டிலும் கூடுதல் தகவல்களைக் காணக்கூடும். மேலும் விவரங்களுக்கு [Bans & Appeals](/guides/bans-appeals/) வழிகாட்டியைப் பார்க்கவும்.

## நீங்கள் நிர்வகிக்கக்கூடியவை

[Profile Settings](/guides/profile-settings/) பகுதியில்:

- Display Name மற்றும் Handle
- Avatar (படத்தை Upload செய்வது அல்லது URL மூலம் அமைப்பது)
- Social Links (TikTok, X போன்றவை)
- YouTube மற்றும் Twitch Visibility Toggle

[Connections](/guides/connections/) பகுதியில்:

- இணைக்கப்பட்ட OAuth Providers (Steam, Discord, Google போன்றவை)

## Badges மற்றும் Programs

Contributor Badge அல்லது Program Awards போன்ற அங்கீகார Badge-கள், Staff அல்லது தானியங்கி (Automated) Program-கள் மூலம் வழங்கப்படும். அதிகாரப்பூர்வ Badge-களை நீங்கள் சுயமாக (Self-Assign) சேர்க்க முடியாது.

---

## Report செய்வது

உள்நுழைந்த (Signed-in) பயனர்கள், ஒரு Profile-ஐ Moderation Review-க்காக Report செய்யலாம். ஒவ்வொரு Report-மும் ஒரு [Support Ticket](/guides/support-tickets/) உருவாக்கும். Report செய்யும்போது உண்மையான மற்றும் துல்லியமான தகவல்களை மட்டுமே வழங்கவும்.

## API

Public Profile தகவல்களை வெளிப்புற Integrations பயன்படுத்த முடியும்:

```
GET https://api.trucklinemp.com/users/{handleOrId}
```

OAuth-இன் `profile` Scope, அனுமதி வழங்கப்பட்ட பயன்பாடுகளுக்கு (Authorized Apps) குறிப்பிட்ட தகவல்களின் ஒரு பகுதியை மட்டும் வழங்கும். மேலும் விவரங்களுக்கு [OAuth Apps](/guides/developers/oauth-apps/) வழிகாட்டியைப் பார்க்கவும்.
