# VTC பொதுவான அமைப்புகள்

General Settings பக்கம் என்பது உங்கள் VTC-யின் முக்கிய அடையாளத்தையும், பொதுமக்களுக்கு காண்பிக்கப்படும் தகவல்களையும் நிர்வகிக்கும் இடமாகும்.

## General Settings-ஐ அணுகுவது

1. உங்கள் VTC-யின் Management Panel-க்கு செல்லவும்.
2. Sidebar-இல் இருந்து **General Settings** என்பதைத் தேர்வு செய்யவும்.
3. **Identity**, **Requirements**, **Social Links**, மற்றும் **Discord Server** ஆகிய பிரிவுகள் காணப்படும்.

---

## VTC அடையாளம் (Identity)

### VTC அடையாளம் (Identity)
- தளம் முழுவதும் காணப்படும் உங்கள் VTC-யின் பெயர்.
- கட்டாயமான புலம்.
- இதை மாற்றினால் Directory மற்றும் Profile-களில் உங்கள் VTC எவ்வாறு காட்டப்படுகிறது என்பது மாற்றப்படும்.

### VTC Handle
- உங்கள் VTC URL-க்கான தனித்துவமான அடையாளம்: `trucklinemp.com/vtc/@yourhandle`
- அதிகபட்சம் 15 எழுத்துகள்.
- சிறிய ஆங்கில எழுத்துக்கள் (lowercase), எண்கள் மற்றும் underscore (`_`) மட்டும் அனுமதிக்கப்படும்.
- Interface-இல் `@` முன்னொட்டுடன் (Prefix) காட்டப்படும்.
- இதை மாற்றினால் உங்கள் VTC-யின் நிரந்தர URL மாறும்.

### முழக்கம்
- உங்கள் VTC Profile-இல் காண்பிக்கப்படும் சுருக்கமான வாசகம்.
- அதிகபட்சம் 60 எழுத்துகள்.
- Profile Header-இல் VTC பெயருக்குக் கீழே காட்டப்படும்.

### விளக்கம்
- உங்கள் VTC பற்றிய விரிவான விளக்கம் (Markdown ஆதரவு உள்ளது).
- அதிகபட்சம் 5,000 எழுத்துகள்.
- கட்டாயமான புலம்.
- உங்கள் VTC-யின் Public Profile-இல் காட்டப்படும்.
- உங்கள் VTC-யின் வரலாறு, நோக்கம் மற்றும் தனித்துவத்தை விளக்க இதைப் பயன்படுத்தவும்.

### விதிகள்
- உங்கள் VTC-யின் விதிமுறைகள் மற்றும் வழிகாட்டுதல்கள் (Markdown ஆதரவு உள்ளது).
- கட்டாயமான புலம்.
- உறுப்பினர்கள் மற்றும் விண்ணப்பதாரர்களுக்கு காண்பிக்கப்படும்.
- விதிமுறைகளை தெளிவாகவும் முழுமையாகவும் எழுதுங்கள்.

### Logo
- உங்கள் VTC-க்கான Profile Picture URL.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.
- Directory, Profile மற்றும் News Feed-களில் காட்டப்படும்.
- பரிந்துரை: சதுர வடிவ படம் (Square), குறைந்தது **256×256px**.

### பேனர்
- உங்கள் VTC Profile-க்கான Cover Banner URL.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.
- Profile Page-இன் மேற்பகுதியில் காட்டப்படும்.
- பரிந்துரை: அகலமான படம் (Wide Image), குறைந்தது **1920×400px**.

## தேவைகள் & ஆட்சேர்ப்பு (Requirements & Recruitment)

உங்கள் VTC-யில் யார் சேரலாம் மற்றும் ஆட்சேர்ப்பு எவ்வாறு நடைபெற வேண்டும் என்பதை இங்கு அமைக்கலாம்.

### கணக்கின் குறைந்தபட்ச வயது
- விண்ணப்பிப்பதற்கு தேவையான TrucklineMP கணக்கின் குறைந்தபட்ச வயதை (நாட்களில்) அமைக்கவும்.
- புதிய கணக்குகளை வடிகட்ட உதவும்.
- அனைத்து கணக்குகளையும் அனுமதிக்க **0** என அமைக்கவும்.

### குறைந்தபட்ச விளையாடும் நேரம்
- விண்ணப்பிக்க தேவையான குறைந்தபட்ச விளையாட்டு நேரத்தை (மணிநேரங்களில்) அமைக்கவும்.
- விண்ணப்பதாரர்களுக்கு போதுமான விளையாட்டு அனுபவம் இருப்பதை உறுதிசெய்ய உதவும்.
- எந்த Playtime-யும் தேவையில்லை என்றால் **0** என அமைக்கவும்.

### முதன்மை மொழி
- உங்கள் VTC-யின் முதன்மை மொழி.
- விருப்பங்கள்: English, German, French, Spanish, Polish.
- உங்கள் VTC Profile-இல் காட்டப்படும்.
- பயனர்கள் தங்களின் மொழிக்கு ஏற்ற VTC-களை கண்டறிய உதவும்.

### ஆள்சேர்ப்பு நடைபெறுகிறது
- ஆட்சேர்ப்பை இயக்க அல்லது முடக்க பயன்படும் Toggle.
- இயக்கப்பட்டால் உங்கள் VTC Directory-இல் **"Hiring"** எனக் காட்டப்படும்.
- முடக்கப்பட்டால் **"Closed"** எனக் காட்டப்படும்.
- தற்போதைய உறுப்பினர்களுக்கு எந்த மாற்றமும் ஏற்படாது.

## சமூக இணைப்புகள் & ஆன்லைன் இருப்பு (Social & Presence)

உங்கள் VTC-யின் ஆன்லைன் தளங்களுக்கான இணைப்புகளைச் சேர்க்கலாம். அனைத்து URL-களும் சரியான `https://` இணைப்பாக இருக்க வேண்டும்.

### Discord Invite
- உங்கள் VTC Discord Server-க்கான அழைப்பு (Invite) இணைப்பு.
- முடிந்தவரை நிரந்தர Invite Link-ஐ பயன்படுத்தவும்.
- உங்கள் VTC Profile-இல் காட்டப்படும்.

### இணையதள முகவரி
- உங்கள் VTC-யின் அதிகாரப்பூர்வ இணையதளம்.
- உங்கள் VTC Profile-இல் காட்டப்படும்.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.

### பொது மின்னஞ்சல்
- உங்கள் VTC-யின் தொடர்பு மின்னஞ்சல்.
- உங்கள் Profile-இல் பொதுவாகக் காட்டப்படும்.
- சரியான Email முகவரியாக இருக்க வேண்டும்.

### Instagram
- உங்கள் VTC-யின் Instagram Profile இணைப்பு.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.

### Twitter/X
- உங்கள் VTC-யின் Twitter அல்லது X Profile இணைப்பு.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.

### Facebook
- உங்கள் VTC-யின் Facebook Page இணைப்பு.
- சரியான HTTPS URL ஆக இருக்க வேண்டும்.

## சிறந்த நடைமுறைகள் (Best Practices)

### சுயவிவரத்தை முழுமையாக்குதல்
- தொழில்முறை தோற்றத்திற்காக அனைத்து புலங்களையும் நிரப்பவும்.
- Logo மற்றும் Banner-க்கு உயர்தர படங்களை பயன்படுத்தவும்.
- உங்கள் VTC-யை சிறப்பாக அறிமுகப்படுத்தும் விளக்கத்தை எழுதவும்.
- விதிமுறைகளை தெளிவாகவும் முழுமையாகவும் வைத்திருக்கவும்.

### URLs
- முடிந்தவரை நிரந்தர இணைப்புகளைப் பயன்படுத்தவும்.
- சேமித்த பிறகு அனைத்து இணைப்புகளையும் சோதிக்கவும்.
- HTTPS URL-களை மட்டும் பயன்படுத்தவும்.

### ஆட்சேர்ப்பு அமைப்புகள்
- பொருத்தமான விண்ணப்பதாரர்களைத் தேர்வு செய்ய நியாயமான தேவைகளை அமைக்கவும்.
- உறுப்பினர்களைத் தேடிக்கொண்டிருக்கும் போது Recruitment-ஐ திறந்த நிலையில் வைத்திருக்கவும்.
- உங்கள் சமூகத்திற்கு ஏற்ப Primary Language-ஐ புதுப்பிக்கவும்.

## சிக்கல் தீர்வு (Troubleshooting)


### "என்னால் மாற்றங்களைச் சேமிக்க முடியவில்லை"
- Name, Description மற்றும் Rules உள்ளிட்ட கட்டாயப் புலங்கள் அனைத்தும் நிரப்பப்பட்டுள்ளனவா என்பதை உறுதிப்படுத்தவும்.
- அனைத்து URL-களும் சரியான HTTPS Link ஆக உள்ளனவா என்பதைச் சரிபார்க்கவும்.
- Email முகவரி சரியான வடிவத்தில் உள்ளதா என்பதை உறுதிப்படுத்தவும்.

### "என்னுடைய படங்கள் காட்டப்படவில்லை"
- URL-கள் பொதுமக்கள் அணுகக்கூடியதாக உள்ளனவா என்பதை உறுதிப்படுத்தவும்.
- நேரடி Image Link-களை பயன்படுத்தவும் (Page Link அல்ல).
- URL `https://` என்று தொடங்குகிறதா என்பதைச் சரிபார்க்கவும்.

### "என்னுடைய VTC Handle-ஐ மாற்ற விரும்புகிறேன்"
- Handle-ஐ General Settings-இல் மாற்றலாம்.
- இதனால் உங்கள் VTC-யின் URL மாறும்.
- பழைய URL-கள் தானாக Redirect செய்யப்படமாட்டாது.
