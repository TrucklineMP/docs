# Regional and Timezone

Events, Notifications மற்றும் நேரம் தொடர்பான பிற அம்சங்களில் தேதிகள் மற்றும் நேரங்களை உங்கள் உள்ளூர் நேரத்திற்கேற்ப (Local Time) காட்ட, TrucklineMP உங்கள் Timezone அமைப்பைப் பயன்படுத்துகிறது.

## Regional Settings-ஐ அணுகுவது

1. மேல் வலது மூலையில் உள்ள உங்கள் Avatar-ஐ கிளிக் செய்யவும்.
2. **Settings** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Regional** தாவலைத் திறக்கவும்.
4. அல்லது `trucklinemp.com/user/me/settings?tab=regional` முகவரிக்குச் செல்லவும்.

---

## Timezone-ஐ தேர்வு செய்வது

நீங்கள் வசிக்கும் இடம் அல்லது Convoy-களை திட்டமிடும் பகுதிக்கு பொருந்தும் IANA Timezone-ஐத் தேர்ந்தெடுக்கவும். உதாரணங்கள்:

- `Europe/London`
- `America/New_York`
- `Europe/Berlin`

உறுதிப்படுத்தியவுடன் (Confirm) மாற்றங்கள் தானாகச் சேமிக்கப்படும். "**2 hours ago**" போன்ற Relative Time-கள், அடுத்த முறை பக்கம் Reload செய்யப்பட்டபோது புதுப்பிக்கப்படும்.

---

## உங்கள் Timezone பயன்படுத்தப்படும் அம்சங்கள்

| அம்சம் | விளைவு |
|---------|--------|
| **Events** | தொடக்க மற்றும் முடிவு நேரங்கள் உங்கள் உள்ளூர் Timezone-இல் காட்டப்படும் |
| **Notifications** | Notification Bell Popover-இல் நேர முத்திரைகள் (Timestamps) காட்டப்படும் |
| **Profile Activity** | ஆதரிக்கப்படும் இடங்களில் உள்ளூர் நேரத்திற்கேற்ப (Localized) காண்பிக்கப்படும் |
| **Forum** | Forum உங்கள் Account Preference-ஐ பயன்படுத்தினால், தேதி மற்றும் நேரம் அதன்படி காட்டப்படும் |

சில VTC கருவிகள் (உதாரணமாக Announcement **Quiet Hours**) உங்கள் தனிப்பட்ட Timezone-ஐ அல்ல, Webhook-இல் அமைக்கப்பட்டுள்ள Timezone-ஐ பயன்படுத்தும். மேலும் விவரங்களுக்கு [VTC Announcements](/guides/vtc-announcements/) வழிகாட்டியைப் பார்க்கவும்.

## சிறந்த நடைமுறைகள்

- நீங்கள் உண்மையில் Event அல்லது Convoy-களை திட்டமிடும் பகுதிக்கான Timezone-ஐத் தேர்ந்தெடுக்கவும்; அவசியம் உங்கள் உடல்நிலை இருப்பிடத்தின் Timezone ஆக இருக்க வேண்டியதில்லை.
- நீங்கள் வேறு இடத்திற்கு குடிபெயர்ந்தாலோ அல்லது உங்கள் முக்கிய விளையாட்டு நேரம் மாறியாலோ, உங்கள் Timezone-ஐ புதுப்பிக்கவும்.
- Event உருவாக்குபவர்கள், Event Editor-இல் எப்போதும் சரியான UTC நேரத்தைப் பயன்படுத்த வேண்டும். பார்வையாளர்களுக்கு அது அவர்களின் உள்ளூர் நேரமாக தானாக மாற்றிக் காட்டப்படும்.

## சிக்கல் தீர்வு

### நேரம் ஒரு மணி நேரம் தவறாகக் காட்டப்படுகிறது

Daylight Saving Time மாற்றங்கள் IANA Timezone-களின் அடிப்படையில் தானாக நிர்வகிக்கப்படுகின்றன. நீங்கள் சமீபத்தில் வேறு இடத்திற்கு பயணம் செய்திருந்தால், உங்கள் Timezone-ஐ மீண்டும் தேர்ந்தெடுக்கவும்.

### Event நேரம் Discord-இல் வேறுபடுகிறது

Discord Messages மற்றும் TrucklineMP Event Pages வெவ்வேறு நேரக் காட்சி முறைகளைப் பயன்படுத்தக்கூடும். அதிகாரப்பூர்வ RSVP நேரத்திற்காக எப்போதும் **trucklinemp.com**-இல் உள்ள Event Page-ஐ நம்பவும்.
