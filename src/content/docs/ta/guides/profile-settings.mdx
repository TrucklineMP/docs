# சுயவிவர அமைப்புகள்

TrucklineMP-இல் மற்ற பயனர்கள் உங்களை எவ்வாறு பார்க்கிறார்கள் என்பதை நிர்வகிக்கவும். உங்கள் Profile என்பது இந்த தளத்தில் உங்கள் பொது அடையாளமாக (Public Identity) செயல்படுகிறது.

## Profile Settings-ஐ அணுகுவது

1. மேல் வலது மூலையில் உள்ள உங்கள் Avatar-ஐ கிளிக் செய்யவும்.
2. **Settings** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Profile** தாவலைத் தேர்ந்தெடுக்கவும்.
4. அல்லது `trucklinemp.com/user/me/settings?tab=profile` முகவரிக்குச் செல்லவும்.

---

## Display Name

அனைத்து பயனர்களுக்கும் காண்பிக்கப்படும் உங்கள் பொது Display Name.

- **நீளம்:** 2–32 எழுத்துகள்
- **Cooldown:** ஒரு நாளுக்கு ஒருமுறை மட்டுமே மாற்ற முடியும்
- **Visibility:** உங்கள் Profile, VTC-கள், Events, Forums மற்றும் News Feed-களில் காண்பிக்கப்படும்

### குறிப்புகள்
- எளிதில் அடையாளம் காணக்கூடிய பெயரைத் தேர்ந்தெடுக்கவும்.
- தட்டச்சு செய்ய சிரமமான Special Characters-ஐ தவிர்க்கவும்.
- அனைத்து தளங்களிலும் ஒரே பெயரைப் பயன்படுத்த முயற்சிக்கவும்.

---

## Social Handle

உங்கள் Profile URL-க்கான தனித்துவமான Handle.

- **URL:** `trucklinemp.com/@yourhandle`
- **நீளம்:** 2–15 எழுத்துகள்
- **அனுமதிக்கப்படுபவை:** சிறிய ஆங்கில எழுத்துகள் (Lowercase Letters), எண்கள் மற்றும் Underscore (`_`) மட்டும்
- **Cooldown:** 30 நாட்களுக்கு ஒருமுறை மட்டுமே மாற்ற முடியும்

### குறிப்புகள்
- சுருக்கமாகவும் எளிதில் நினைவில் நிற்கக்கூடியதாகவும் வைத்திருக்கவும்.
- உங்கள் பிற Social Media Handle-களுடன் பொருந்துமாறு பயன்படுத்தவும்.
- Spam போல தோன்றக்கூடிய தேவையற்ற எண்களைத் தவிர்க்கவும்.

## Profile Picture

தளம் முழுவதும் காண்பிக்கப்படும் உங்கள் Avatar.

### புதிய படத்தை பதிவேற்றுவது
1. **Upload picture** என்பதைக் கிளிக் செய்யவும்.
2. ஒரு Image File-ஐத் தேர்ந்தெடுக்கவும் (JPEG, PNG அல்லது WebP).
3. அதிகபட்ச File Size: **5 MB**
4. படம் உடனடியாக பதிவேற்றப்பட்டு பயன்படுத்தப்படும்.

### Steam-இலிருந்து ஒத்திசைப்பது
1. **Sync from Steam** என்பதைக் கிளிக் செய்யவும்.
2. உங்கள் Steam Account-இலிருந்து Profile Picture புதுப்பிக்கப்படும்.
3. தற்போதைய Steam Avatar தானாகப் பெறப்படும்.

### குறிப்புகள்
- தெளிவாகவும் எளிதில் அடையாளம் காணக்கூடிய படத்தைப் பயன்படுத்தவும்.
- சதுர (Square) வடிவிலான படங்கள் சிறந்தவை.
- அதிக விவரங்கள் கொண்ட பின்னணிகளைத் தவிர்க்கவும்.
- சிறிய அளவிலும் உங்கள் முகம் அல்லது Logo தெளிவாகத் தெரியுமாறு இருக்க வேண்டும்.

## Social Links

உங்கள் சமூக வலைதள Profile இணைப்புகளைச் சேர்க்கலாம். இவை உங்கள் Public Profile-இல் காண்பிக்கப்படும்.

### TikTok
- உங்கள் முழு TikTok Profile URL-ஐ உள்ளிடவும்.
- உதாரணம்: `https://www.tiktok.com/@yourname`
- சரியான TikTok URL ஆக இருக்க வேண்டும்.

### World of Trucks
- உங்கள் World of Trucks Profile URL-ஐ உள்ளிடவும்.
- உதாரணம்: `https://worldoftrucks.com/en/profile/...`
- சரியான World of Trucks URL ஆக இருக்க வேண்டும்.

### Twitter / X
- உங்கள் Twitter அல்லது X Profile URL-ஐ உள்ளிடவும்.
- உதாரணம்: `https://twitter.com/yourhandle` அல்லது `https://x.com/yourhandle`
- சரியான Twitter/X URL ஆக இருக்க வேண்டும்.

### Medal
- உங்கள் Medal Profile URL-ஐ உள்ளிடவும்.
- உதாரணம்: `https://medal.tv/u/yourname`
- சரியான Medal URL ஆக இருக்க வேண்டும்.

### URL Validation
- அனைத்து இணைப்புகளும் `https://` எனத் தொடங்க வேண்டும்.
- URL-கள் சரியான Domain-ஐ அடிப்படையாகக் கொண்டு சரிபார்க்கப்படும்.
- தவறான URL கொடுக்கப்பட்டால் Error Message காண்பிக்கப்படும்.
- `https://` இல்லாமல் URL உள்ளிட்டால், அது தானாகவே சேர்க்கப்படும்.

## மாற்றங்களைச் சேமிப்பது

1. Form-இல் தேவையான மாற்றங்களைச் செய்யவும்.
2. கீழே உள்ள **Save Changes** என்பதைக் கிளிக் செய்யவும்.
3. வெற்றிகரமான Success Notification காண்பிக்கப்படும்.
4. மாற்றங்கள் உடனடியாக உங்கள் Profile-இல் பிரதிபலிக்கும்.

## சிக்கல் தீர்வு

### "என்னால் என் பெயரை மாற்ற முடியவில்லை"
- Display Name-ஐ ஒரு நாளுக்கு ஒருமுறை மட்டுமே மாற்ற முடியும்.
- பெயர் 2–32 எழுத்துகளுக்குள் உள்ளதா என்பதை உறுதிசெய்யவும்.
- அனுமதிக்கப்படாத எழுத்துகள் உள்ளனவா என்பதைச் சரிபார்க்கவும்.

### "என்னுடைய Handle சேமிக்கப்படவில்லை"
- Handle 2–15 எழுத்துகளுக்குள் இருக்க வேண்டும்.
- சிறிய ஆங்கில எழுத்துகள், எண்கள் மற்றும் Underscore (`_`) மட்டும் பயன்படுத்தலாம்.
- Handle-ஐ 30 நாட்களுக்கு ஒருமுறை மட்டுமே மாற்ற முடியும்.

### "என்னுடைய Profile Picture பதிவேற்றப்படவில்லை"
- File Format (JPEG, PNG அல்லது WebP) சரியானதா என்பதைச் சரிபார்க்கவும்.
- File Size 5 MB-க்கு குறைவாக இருக்க வேண்டும்.
- வேறு ஒரு படத்தை முயற்சிக்கவும்.

### "என்னுடைய Social Links-ல் Error வருகிறது"
- URL `https://` எனத் தொடங்குகிறதா என்பதை உறுதிசெய்யவும்.
- URL சரியான Platform-இற்கானதா என்பதைச் சரிபார்க்கவும்.
- தேவையற்ற இடைவெளிகள் (Spaces) அல்லது கூடுதல் எழுத்துகளை நீக்கவும்.
