# Account Security

உங்கள் கணக்கைப் பாதுகாப்பாக வைத்திருக்க, கணக்கு பாதுகாப்பு, செயலில் உள்ள அமர்வுகள் மற்றும் தனியுரிமை அமைப்புகளை நிர்வகிக்கவும்.

## Security Settings-ஐ அணுகுவது

1. மேல் வலது மூலையில் உள்ள உங்கள் **Avatar**-ஐ கிளிக் செய்யவும்.
2. **Settings** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Security** Tab-ஐத் தேர்ந்தெடுக்கவும்.
4. அல்லது `trucklinemp.com/user/me/settings?tab=security` முகவரிக்குச் செல்லவும்.

---

## இணைக்கப்பட்ட கணக்கு (Steam)

உங்கள் Steam கணக்கு, உங்கள் TrucklineMP கணக்குடன் நிரந்தரமாக இணைக்கப்பட்டுள்ளது.

- **நிலை:** துண்டிக்க முடியாது
- **நோக்கம்:** முதன்மை அடையாளச் சரிபார்ப்பு
- **Steam ID:** இந்தப் பகுதியில் காட்டப்படும்

இந்த இணைப்பு கணக்கு சரிபார்ப்பிற்குத் தேவையானது; இதை மாற்ற முடியாது.

---

## இரு-அடுக்கு அங்கீகாரம் (2FA)

உங்கள் கணக்கிற்கு கூடுதல் பாதுகாப்பு அடுக்கைச் சேர்க்கவும்.

### 2FA-ஐ செயல்படுத்துவது

1. **Enable 2FA** என்பதைக் கிளிக் செய்யவும்.
2. ஒரு QR குறியீடு உருவாக்கப்படும்.
3. கீழே உள்ள அங்கீகார பயன்பாடுகளில் ஏதேனும் ஒன்றைப் பயன்படுத்தி QR குறியீட்டை ஸ்கேன் செய்யவும்:
   - Google Authenticator
   - Authy
   - Microsoft Authenticator
   - TOTP-ஐ ஆதரிக்கும் ஏதேனும் ஒரு பயன்பாடு
4. அல்லது QR குறியீட்டின் கீழே காட்டப்படும் இரகசிய விசையை (Secret Key) கைமுறையாக உள்ளிடவும்.
5. **Continue to Verification** என்பதைக் கிளிக் செய்யவும்.
6. உங்கள் அங்கீகார பயன்பாட்டில் காணப்படும் 6 இலக்கக் குறியீட்டை உள்ளிடவும்.
7. **Verify & Enable** என்பதைக் கிளிக் செய்யவும்.
8. **உங்கள் Backup Codes-ஐ பாதுகாப்பாகச் சேமிக்கவும்** — அவை மீண்டும் காட்டப்படமாட்டாது.

### Backup Codes

2FA-ஐ செயல்படுத்திய பிறகு, உங்களுக்கு **10 Backup Codes** வழங்கப்படும்:
- ஒவ்வொரு குறியீட்டையும் **ஒரு முறை மட்டுமே** பயன்படுத்த முடியும்.
- உங்கள் அங்கீகார பயன்பாட்டை அணுக முடியாத சூழலில் இவற்றைப் பயன்படுத்தலாம்.
- இவற்றைப் பாதுகாப்பான இடத்தில் (Password Manager அல்லது அச்சிடப்பட்ட நகல்) சேமிக்கவும்.
- அனைத்து குறியீடுகளையும் ஒரே நேரத்தில் நகலெடுக்கலாம்.

### 2FA-ஐ முடக்குவது

1. **Disable 2FA** என்பதைக் கிளிக் செய்யவும்.
2. உங்கள் அங்கீகாரக் குறியீடு **அல்லது** ஒரு Backup Code-ஐ உள்ளிடவும்.
3. **Disable 2FA** என்பதைக் கிளிக் செய்யவும்.
4. 2FA உங்கள் கணக்கிலிருந்து அகற்றப்படும்.

**எச்சரிக்கை:** 2FA-ஐ முடக்குவது உங்கள் கணக்கின் பாதுகாப்பைக் குறைக்கும்.

### உங்கள் Authenticator பயன்பாட்டை இழந்தால்

1. ஒரு Backup Code-ஐ பயன்படுத்தி உள்நுழையவும்.
2. உள்நுழைந்த பிறகு, **Security Settings** பகுதிக்குச் செல்லவும்.
3. Backup Code-ஐ பயன்படுத்தி 2FA-ஐ முடக்கவும்.
4. புதிய Authenticator பயன்பாட்டைப் பயன்படுத்தி மீண்டும் 2FA-ஐ செயல்படுத்தவும்.

**Backup Codes இல்லையெனில்:** உங்கள் அடையாளச் சான்றுடன் Discord மூலம் ஆதரவு குழுவைத் தொடர்புகொள்ளவும்.

## Developer Mode

தொழில்முறை Developer கருவிகளை அணுகுவதற்கு **Developer Console**-ஐ செயல்படுத்தவும்.

### இதன் மூலம் நீங்கள் பெறுவது
- API திட்டங்களை உருவாக்கலாம்.
- API சான்றுகளை (Credentials) நிர்வகிக்கலாம்.
- Developer ஆவணங்களை அணுகலாம்.
- TrucklineMP-உடன் ஒருங்கிணைப்புகளை (Integrations) உருவாக்கலாம்.

### Developer Mode-ஐ செயல்படுத்துவது
1. **Developer Mode** பகுதியில் உள்ள **Activate** என்பதைக் கிளிக் செய்யவும்.
2. உறுதிப்படுத்தல் உரையாடலைப் படிக்கவும்.
3. **Confirm Activation** என்பதைக் கிளிக் செய்யவும்.
4. Developer Mode செயல்படுத்தப்படும்.

**எச்சரிக்கை:** இந்தச் செயல் நிரந்தரமானது; இதை மீண்டும் மாற்ற முடியாது.

### Console-ஐ அணுகுவது
- செயல்படுத்திய பிறகு, **Open Console** என்பதைக் கிளிக் செய்து Developer கருவிகளை அணுகலாம்.
- அல்லது `trucklinemp.com/developer` முகவரிக்குச் செல்லலாம்.

## செயலில் உள்ள அமர்வுகள் (Active Sessions)

தற்போது உங்கள் கணக்கில் உள்நுழைந்துள்ள அனைத்து சாதனங்களையும் பார்வையிட்டு நிர்வகிக்கலாம்.

### அமர்வு விவரங்கள்
ஒவ்வொரு அமர்விலும் பின்வரும் தகவல்கள் காணப்படும்:
- **Browser** — Chrome, Firefox, Safari, Edge
- **Operating System** — Windows, macOS, Linux, Android, iOS
- **IP Address** — அந்த அமர்வு தொடங்கப்பட்ட IP முகவரி
- **Start Date** — அமர்வு உருவாக்கப்பட்ட தேதி
- **Current Badge** — தற்போது பயன்படுத்திக் கொண்டிருக்கும் அமர்வைக் குறிக்கும்

### ஒரு அமர்வை நீக்குவது
1. நீக்க விரும்பும் அமர்வைக் கண்டறியவும்.
2. அதற்கு அருகிலுள்ள **Revoke** என்பதைக் கிளிக் செய்யவும்.
3. அந்த அமர்வு உடனடியாக வெளியேற்றப்படும்.
4. அந்தச் சாதனம் மீண்டும் உள்நுழைய வேண்டும்.

### மற்ற அனைத்து அமர்வுகளிலிருந்தும் வெளியேறுவது
1. மேலே உள்ள **Sign out all other sessions** என்பதைக் கிளிக் செய்யவும்.
2. தற்போதைய அமர்வைத் தவிர மற்ற அனைத்தும் முடிவுக்குக் கொண்டுவரப்படும்.
3. அனுமதியில்லாத அணுகல் இருப்பதாக சந்தேகித்தால் இது மிகவும் பயனுள்ளதாக இருக்கும்.

## தனியுரிமை அமைப்புகள் (Privacy Settings)

### வெளிப்புற Configuration-களை மறைப்பது
- இதைச் செயல்படுத்தினால், உங்கள் Truckline Utils Configuration-களை நீங்கள் மற்றும் நிர்வாகிகள் மட்டுமே பார்க்க முடியும்.
- பிற பயனர்கள் உங்கள் Profile-இல் அவற்றைப் பார்க்க முடியாது.
- இதை இயக்க அல்லது முடக்க Checkbox-ஐ பயன்படுத்தவும்.

## சட்ட ஆவணங்கள்

நீங்கள் ஏற்றுக்கொண்டுள்ள சட்ட ஆவணங்களின் நிலையைப் பார்வையிடலாம்:
- சேவை விதிமுறைகள்
- தனியுரிமைக் கொள்கை
- பிற கட்டாய ஒப்பந்தங்கள்

நிலுவையில் உள்ள ஆவணங்கள் பட்டியலிடப்படும்; சில அம்சங்களைத் தொடர்ந்து பயன்படுத்த அவற்றை ஏற்க வேண்டும்.

## பணியாளர் ஆவண உறுதிப்படுத்தல்கள்

நீங்கள் ஒரு Staff உறுப்பினராக இருந்தால், நீங்கள் உறுதிப்படுத்திய ஆவணங்களைப் பார்க்கலாம்:
- ஆவணத்தின் தலைப்பு மற்றும் பதிப்பு
- ஏற்றுக்கொண்ட தேதி

---

## பாதுகாப்பிற்கான சிறந்த நடைமுறைகள்

### மிகவும் அவசியமானவை
- **2FA-ஐ செயல்படுத்தவும்** — உங்கள் கணக்கைப் பாதுகாக்கும் மிக முக்கியமான நடவடிக்கை.
- **Backup Codes-ஐ பாதுகாப்பாகச் சேமிக்கவும்** — Password Manager-ல் வைத்திருப்பது சிறந்தது.
- **அமர்வுகளை தவறாமல் பரிசோதிக்கவும்** — அறிமுகமில்லாத அமர்வுகளை உடனடியாக நீக்கவும்.
- **வலுவான கடவுச்சொல்லைப் பயன்படுத்தவும்** — TrucklineMP-க்கென தனித்துவமான கடவுச்சொல்லைப் பயன்படுத்தவும்.

### பரிந்துரைக்கப்படுவது
- பகிரப்பட்ட சாதனங்களில் இருந்து வெளியேறவும்.
- அனுமதிக்கப்பட்ட பயன்பாடுகளை அவ்வப்போது பரிசோதிக்கவும்.
- இணைக்கப்பட்ட கணக்குகளை புதுப்பித்த நிலையில் வைத்திருக்கவும்.
- சந்தேகத்திற்கிடமான செயல்பாடுகளை உடனடியாக புகாரளிக்கவும்.

## சிக்கல் தீர்வு

### "என்னால் 2FA-ஐ செயல்படுத்த முடியவில்லை"
- உங்கள் Authenticator பயன்பாடு TOTP-ஐ ஆதரிக்கிறதா என்பதை உறுதிசெய்யவும்.
- உங்கள் சாதனத்தின் நேரம் சரியாக ஒத்திசைக்கப்பட்டுள்ளதா என்பதைச் சரிபார்க்கவும்.
- வேறு ஒரு Authenticator பயன்பாட்டைப் பயன்படுத்திப் பார்க்கவும்.

### "எனது Backup Codes-ஐ இழந்துவிட்டேன்"
- இன்னும் உள்நுழைந்திருந்தால், 2FA-ஐ முடக்கி மீண்டும் செயல்படுத்தவும்.
- உள்நுழைய முடியாவிட்டால், Discord மூலம் ஆதரவு குழுவைத் தொடர்புகொள்ளவும்.
- கணக்கு மீட்பிற்காக உங்கள் அடையாளச் சான்றை வழங்கவும்.

### "எனக்குத் தெரியாத ஒரு அமர்வைக் காண்கிறேன்"
- அந்த அமர்வை உடனடியாக **Revoke** செய்யவும்.
- முடிந்தால் உங்கள் கடவுச்சொல்லை மாற்றவும்.
- இதுவரை செய்யவில்லை என்றால் 2FA-ஐ செயல்படுத்தவும்.
- அனுமதியில்லாத அணுகல் இருப்பதாக சந்தேகித்தால் ஆதரவு குழுவைத் தொடர்புகொள்ளவும்.

### "2FA மூலம் என்னால் உள்நுழைய முடியவில்லை"
- உங்கள் Authenticator பயன்பாட்டின் நேரம் சரியாக உள்ளதா என்பதைச் சரிபார்க்கவும்.
- ஒரு Backup Code-ஐ பயன்படுத்திப் பார்க்கவும்.
- எதுவும் செயல்படவில்லை என்றால், ஆதரவு குழுவைத் தொடர்புகொள்ளவும்.
