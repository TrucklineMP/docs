# தோற்றம் & விருப்பத்தேர்வுகள்

TrucklineMP-இன் தோற்றத்தை உங்கள் விருப்பத்திற்கு ஏற்ப தனிப்பயனாக்கி, உங்கள் கணக்கு அமைப்புகளை நிர்வகிக்கவும்.

## தோற்ற அமைப்புகளை அணுகுதல்

1. மேல் வலது மூலையில் உள்ள உங்கள் Avatar-ஐ கிளிக் செய்யவும்.
2. **Settings** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Appearance** தாவலைத் தேர்ந்தெடுக்கவும்.
4. அல்லது `trucklinemp.com/user/me/settings?tab=appearance` முகவரிக்குச் செல்லவும்.

## Themes

TrucklineMP எவ்வாறு தோன்ற வேண்டும் என்பதைத் தேர்வு செய்யுங்கள். நீங்கள் தேர்ந்தெடுக்கும் Theme தானாகவே சேமிக்கப்படும்.

### Available Themes

#### Dark
- இயல்புநிலை (Default) Marathon தோற்றம்.
- கருமை நிற பின்னணியுடன் தெளிவான, தொழில்துறை பாணி வடிவமைப்பு.
- அதிக Contrast கொண்ட எழுத்துருக்கள்.
- குறைந்த வெளிச்ச சூழலில் கண்களுக்கு வசதியானது.

#### Light
- சுத்தமான வெள்ளை பின்னணி.
- அதிக வெளிச்சம் உள்ள சூழல்களுக்கு ஏற்ற உயர்ந்த Contrast.
- தொழில்முறை தோற்றம்.
- நல்ல வெளிச்சம் உள்ள இடங்களில் படிக்க எளிதானது.

#### Awfully Familiar Grey
- பச்சை நிற Accent-களுடன் கூடிய அடர் சாம்பல் நிற Theme.
- வித்தியாசமானதாய் இருந்தாலும் பரிச்சயமான உணர்வை வழங்கும் தோற்றம்.
- மிதமான Contrast.
- Dark மற்றும் Light Theme-களுக்கு இடையிலான சமநிலை.

### Theme-ஐ தேர்ந்தெடுப்பது
1. Theme Card-களை உலாவவும்.
2. ஒவ்வொரு Card-லும் அந்த Theme-இன் நிறங்களை காட்டும் Preview இருக்கும்.
3. பயன்படுத்த விரும்பும் Theme-ஐ கிளிக் செய்யவும்.
4. மாற்றம் உடனடியாக செயல்படுத்தப்பட்டு தானாகவே சேமிக்கப்படும்.
5. தற்போது பயன்படுத்தப்படும் Theme-இல் சிறிய Indicator Dot காணப்படும்.

## Regional Settings

உங்கள் பிராந்தியத்திற்கான (Regional) விருப்பங்களை அமைக்கவும்.

### Regional Settings-ஐ அணுகுவது
1. **Settings** > **Regional** தாவலுக்குச் செல்லவும்.
2. அல்லது `trucklinemp.com/user/me/settings?tab=regional` முகவரிக்குச் செல்லவும்.

## Authorized Apps

உங்கள் கணக்கை அணுக அனுமதி பெற்றுள்ள Third-Party பயன்பாடுகளை நிர்வகிக்கவும்.

### Authorized Apps-ஐ அணுகுவது
1. **Settings** > **Authorized Apps** தாவலுக்குச் செல்லவும்.
2. அல்லது `trucklinemp.com/user/me/settings?tab=authorized-apps` முகவரிக்குச் செல்லவும்.

### Authorized Apps என்றால் என்ன?
OAuth மூலம் உங்கள் TrucklineMP கணக்கை அணுக அனுமதி வழங்கப்பட்ட Third-Party பயன்பாடுகள்.உதாரணமாக:
- Truck Configurator-கள்
- Event Management Tools
- Community Bots
- Analytics Dashboards

### பயன்பாடுகளைப் பரிசோதித்தல்
ஒவ்வொரு பயன்பாட்டிற்கும் பின்வரும் விவரங்களைக் காணலாம்:
- **Application Name** — Third-Party பயன்பாட்டின் பெயர்.
- **Permissions Granted** — அந்த பயன்பாட்டிற்கு வழங்கப்பட்ட அனுமதிகள்.
- **Date Authorized** — அனுமதி வழங்கப்பட்ட தேதி.

### அணுகலை நீக்குவது
1. நீக்க விரும்பும் பயன்பாட்டைக் கண்டறியவும்.
2. **Revoke** என்பதைக் கிளிக் செய்யவும்.
3. அந்த பயன்பாட்டின் அணுகல் உடனடியாக நீக்கப்படும்.
4. மீண்டும் பயன்படுத்த வேண்டுமெனில், அதற்கு மறுபடியும் அனுமதி வழங்க வேண்டும்.

### எப்போது அணுகலை நீக்க வேண்டும்?
- அந்த பயன்பாட்டை இனி பயன்படுத்தவில்லை.
- அனுமதியில்லாத அணுகல் இருப்பதாக சந்தேகிக்கிறீர்கள்.
- அந்த பயன்பாடு பாதுகாப்பு சிக்கலுக்கு உள்ளாகியுள்ளது.
- பழைய Connections-ஐ சுத்தப்படுத்த விரும்புகிறீர்கள்.

## Data Export

TrucklineMP-இல் சேமிக்கப்பட்டுள்ள உங்கள் தரவின் நகலை கோருங்கள்.

### Data Export-ஐ அணுகுவது
1. **Settings** > **Data Export** தாவலுக்குச் செல்லவும்.
2. அல்லது `trucklinemp.com/user/me/settings?tab=export` முகவரிக்குச் செல்லவும்.

### இதில் என்ன இருக்கும்?
உங்கள் Export-இல் பின்வருவன இடம்பெறும்:
- Profile தகவல்கள் (பெயர், Handle, Profile Picture, சமூக வலைதள இணைப்புகள்)
- VTC உறுப்பினர் வரலாறு
- Activity Logs
- Configuration Data
- Forum பதிவுகள் மற்றும் Threads
- Event பங்கேற்பு பதிவுகள்

### Data Export-ஐ கோருவது
1. **Request Export** என்பதைக் கிளிக் செய்யவும்.
2. உங்கள் கோரிக்கை செயலாக்க வரிசையில் (Queue) சேர்க்கப்படும்.
3. Export தயாரானதும் Email மூலம் அறிவிக்கப்படும்.
4. Email-இல் உள்ள இணைப்பைப் பயன்படுத்தி தரவைக் பதிவிறக்கவும்.
5. பதிவிறக்க இணைப்பு குறிப்பிட்ட காலத்திற்கு மட்டுமே செல்லுபடியாகும்.

### தரவு வடிவம்(Data Format)
- Export-கள் JSON வடிவத்தில் வழங்கப்படும்.
- அனைத்து தரவுகளும் Machine-Readable வடிவத்தில் இருக்கும்.
- இதை பிற கருவிகளில் பயன்படுத்தலாம் அல்லது உங்கள் பதிவுகளுக்காக பாதுகாத்து வைத்துக்கொள்ளலாம்.

## மன்ற அமைப்புகள்(Forum Settings)

உங்கள் Forum விருப்பங்களை அமைக்கவும்.

### Forum Settings-ஐ அணுகுவது
1. **Settings** > **Forum** தாவலுக்குச் செல்லவும்.
2. அல்லது `trucklinemp.com/user/me/settings?tab=forum` முகவரிக்குச் செல்லவும்.

### கிடைக்கும் அமைப்புகள்
- **Notification Preferences** — Forum அறிவிப்புகளை எவ்வாறு பெற வேண்டும் என்பதை நிர்வகிக்கலாம்.
- **Display Options** — Forum உள்ளடக்கம் எவ்வாறு காட்டப்பட வேண்டும் என்பதைத் தனிப்பயனாக்கலாம்.
- **Signature Settings** — Forum பதிவுகளுக்கான உங்கள் Signature-ஐ அமைக்கலாம்.

## சிறந்த நடைமுறைகள்

### Themes
- நீங்கள் பொதுவாக பயன்படுத்தும் வெளிச்ச சூழலுக்கு ஏற்ற Theme-ஐ தேர்ந்தெடுக்கவும்.
- இரவு நேர ஓட்டங்களுக்கு Dark Theme சிறந்தது.
- பகல் நேர பயன்பாட்டிற்கு Light Theme சிறந்தது.

### Authorized Apps
- அனுமதி வழங்கப்பட்ட பயன்பாடுகளை அவ்வப்போது பரிசோதிக்கவும்.
- பயன்படுத்தாத பயன்பாடுகளின் அணுகலை நீக்கவும்.
- நம்பகமான பயன்பாடுகளுக்கு மட்டுமே அனுமதி வழங்கவும்.
- அனுமதி வழங்குவதற்கு முன் Permissions-ஐ கவனமாகப் பரிசோதிக்கவும்.

### Data Export
- கணக்கை நீக்கும் முன் Data Export-ஐ கோரவும்.
- உங்கள் தரவின் Backup-ஐ பாதுகாப்பாக வைத்திருக்கவும்.
- எந்த தரவுகள் சேமிக்கப்பட்டுள்ளன என்பதை அறிய அவற்றைப் பரிசோதிக்கவும்.

## சிக்கல் தீர்வு

### "Theme மாறவில்லை"
- பக்கத்தை Refresh செய்யவும்.
- Browser Cache-ஐ நீக்கவும்.
- வேறு Browser-ஐ பயன்படுத்திப் பார்க்கவும்.

### "ஒரு பயன்பாட்டின் அணுகலை நீக்க முடியவில்லை"
- இணைய இணைப்பு நிலையாக உள்ளதா என்பதை உறுதிசெய்யவும்.
- பக்கத்தை Refresh செய்து மீண்டும் முயற்சிக்கவும்.
- சிக்கல் தொடர்ந்தால் Support-ஐ தொடர்புகொள்ளவும்.

### "என்னுடைய Data Export வரவில்லை"
- உங்கள் Email-இன் Spam Folder-ஐச் சரிபார்க்கவும்.
- Export தயாராக சில நிமிடங்கள் ஆகலாம்.
- ஒரு மணி நேரத்திற்கும் மேலாக தாமதமானால் Support-ஐ தொடர்புகொள்ளவும்.
