# Official Discord Server

TrucklineMP operates an **official Discord community** separate from VTC-owned servers. The official server uses a different bot stack than the public **TrucklineMP bot** used for VTC servers and Linked Roles.

Join via the invite linked from [trucklinemp.com](https://trucklinemp.com) or Discord discovery.

---

## Two bots, two jobs

| Bot | Where | Role |
|-----|-------|------|
| **TrucklineMP bot** (DiscordBot) | VTC and public servers | Linked Roles, `/profile`, `/verify-server`, notification DMs, VTC `/vtc` |
| **Modbot** | Official main, staff, and beta servers | Moderation, tickets, recruitment onboarding, logging |

Do not expect `/verify-server` on the official server for your VTC. Verify on **your** VTC's Discord.

---

## Website support vs Discord tickets

| Need | Where to go |
|------|-------------|
| Ban appeals | [trucklinemp.com/support?lane=appeal](/guides/bans-appeals/) |
| Account issues | [Support tickets](/guides/support-tickets/) on the website |
| Player reports (platform) | Website support or profile report |
| Discord-only chat moderation | Official server ticket panels (in Discord) |

Modbot may direct you to the website for appeals. **Website appeals are authoritative** for platform bans.

---

## Recruitment and staff onboarding

Staff recruitment flows through the official Discord after an application is accepted on the recruitment portal. Steps can include:

- Joining the main guild
- Beta NDA verification on the website
- Role assignment via automated onboarding

Candidates should follow instructions in DMs and channels precisely.

---

## Linked Roles on the official server

The official server may use TrucklineMP Linked Roles for verified members or staff. Setup is the same as any server: [Linked Roles](/guides/discord-linked-roles/).

---

## VTC servers are independent

VTC Discord servers are run by VTC owners. TrucklineMP staff do not moderate VTC servers unless a platform rule violation is reported through [support](/guides/support-tickets/).

Link your VTC server on the website with [VTC Discord Verification](/guides/vtc-discord-verification/).
