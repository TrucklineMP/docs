# Creating a VTC

A Virtual Trucking Company (VTC) is your organization's home on TrucklineMP. This guide walks through eligibility, the creation wizard, and what happens after your VTC goes live.

## Before you start

You need:

- A signed-in TrucklineMP account
- Permission to create a VTC (platform eligibility rules apply)
- No existing VTC where you are already the owner (one VTC ownership per account)

Open the creation flow at [trucklinemp.com/vtc/create](https://trucklinemp.com/vtc/create).

---

## Eligibility check

When you open the create page, TrucklineMP runs an eligibility check automatically. If you are not eligible, the page explains why. Common reasons include:

- Already owning a VTC
- Account standing or platform restrictions
- Feature disabled by platform configuration

You cannot proceed until eligibility passes.

---

## Creation wizard

The wizard has five steps: **Identity**, **Description**, **Rules**, **Details**, and **Review**.

### Step 1: Identity

| Field | Notes |
|-------|-------|
| **VTC Name** | Display name shown across the platform. Required. |
| **Handle** | Unique URL slug: `trucklinemp.com/vtc/@yourhandle`. 2-15 characters, lowercase letters, numbers, and underscores only. Required. |
| **Slogan** | Short tagline (optional). |
| **Primary Language** | Default language for your VTC. |
| **Logo** | HTTPS URL to a square profile image (optional). |
| **Banner** | HTTPS URL to a wide cover image (optional). |
| **Visibility** | Public, unlisted, or private. See [Visibility](/guides/vtc-visibility/). |

### Step 2: Description

Write your VTC story in Markdown. Minimum 20 characters. This appears on your public profile.

### Step 3: Rules

Write your member rules in Markdown. Minimum 20 characters. Applicants and members can read these on your profile.

### Step 4: Details

| Field | Notes |
|-------|-------|
| **Minimum Account Age** | Days a TrucklineMP account must exist before applying. Set `0` for no limit. |
| **Minimum Playtime** | Hours of playtime required to apply. Set `0` for no limit. |
| **Recruitment Open** | Whether applicants can apply immediately after creation. |
| **Contact Email** | Optional public contact address. |
| **Social Links** | Website, Discord invite, Instagram, Facebook, X/Twitter. Must be full `https://` URLs if provided. |

### Step 5: Review

Confirm every field, then submit. On success you are redirected to your new VTC profile.

## Default setup

When your VTC is created, TrucklineMP seeds default roles:

- **Management**
- **Staff**
- **Driver**
- **Recruit**

You become the owner with full permissions. Open **Manage** from your VTC profile to configure recruitment, members, and Discord linking.

## Troubleshooting

### "Not eligible to create a VTC"

Read the reasons shown on the create page. You may already own a VTC or need to resolve a standing issue.

### Handle will not save

Handles must be 2-15 characters, lowercase, and use only letters, numbers, and underscores. The handle must be unique.

### Creation fails on social URLs

Every optional URL must start with `https://`. Discord invite links must be full URLs, not bare codes.
