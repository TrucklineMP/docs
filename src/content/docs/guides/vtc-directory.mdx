# VTC Directory

The VTC directory helps drivers discover Virtual Trucking Companies. Browse all public listings, filter by what matters to you, and open a VTC profile to apply or learn more.

## Accessing the directory

Open [trucklinemp.com/vtc](https://trucklinemp.com/vtc).

Only **public** and **unlisted** VTCs appear here. Private VTCs are hidden unless you have an invite link.

---

## Search and filters

Use the toolbar to narrow results:

| Filter | Purpose |
|--------|---------|
| **Search** | Match VTC name or handle |
| **Verified** | Show only verified VTCs |
| **Official** | Show platform-official VTCs |
| **Recruitment open** | VTCs currently accepting applications |
| **Language** | Primary language filter |

### Sort options

- Newest first
- Name (A-Z)
- Member count

Results update as you change filters. Pagination loads more VTCs when available.

---

## VTC cards

Each card shows:

- Logo and banner preview
- VTC name and handle (`@handle`)
- Slogan or short description excerpt
- Badges (verified, official, partnered, recruitment open)
- Member count
- Primary language

Click a card to open the full VTC profile.

---

## VTC profile overview

From a profile you can:

- Read the full description and rules
- See upcoming events and news
- View member preview and open the full roster
- Apply if recruitment is open
- Join via invite if you have a code
- See a verified Discord server link when configured

Profile URLs use handles: `trucklinemp.com/vtc/@handle`

---

## For VTC owners

To improve discoverability:

1. Set visibility to **public** in [General Settings](/guides/vtc-general-settings/)
2. Keep recruitment open if you are hiring
3. Apply for [Verified](/guides/verified-vtc-program/) or [Partnered](/guides/partnered-vtc-program/) status when eligible
4. Post [news](/guides/vtc-news/) and [events](/guides/vtc-events/) so your profile stays active

Unlisted VTCs are reachable by direct link but do not appear in directory browse results.

---

## API access

Third-party tools can list VTCs through the Public API:

```
GET https://api.trucklinemp.com/vtcs
```

See [Public API](/guides/developers/public-api/) for authentication and pagination.
